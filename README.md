# 🏃‍♂️ Urban Dash

![Urban Dash Banner](https://images.unsplash.com/photo-1614850523296-d8c1af93d400?q=80&w=1200&auto=format&fit=crop)

A high-octane 3D endless runner built with **React**, **Three.js**, and **Tailwind CSS**. Dodge red metros, leap over barricades, and race through a vibrant neon-urban landscape.

## ✨ Features

- **Dynamic 3D World**: An endless, perspective-driven city environment with rolling tracks and atmospheric lighting.
- **Fluid Gameplay Loop**: Rapid lane-switching and gravity-based jumping mechanics.
- **Vibrant Aesthetic**: A custom "Vibrant Palette" theme featuring sunrise gradients, glass-morphism HUD, and neon-lit entities.
- **Rich Collectibles**:
  - 🪙 **Coins**: Standard currency for consistent scoring.
  - 💎 **Gems**: Rare cyan crystals worth 200 points.
  - 💍 **Special Tokens**: Elite purple rings worth 500 points.
- **Adaptive Controls**: Optimized for both high-precision keyboard play and responsive mobile swipe gestures.
- **Arcade Audio**: Retro-synth sound effects for jumps, collections, and crashes.

## 🎮 How to Play

### Desktop
- **A / D** or **Left / Right Arrows**: Switch lanes
- **W / Space** or **Up Arrow**: Jump

### Mobile
- **Swipe Left / Right**: Switch lanes
- **Swipe Up**: Jump

## 🛠 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **3D Engine**: [React Three Fiber](https://r3f.docs.pmnd.rs/) & [Three.js](https://threejs.org/)
- **Animations**: [Motion](https://motion.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1. **Start the Game**: Click the large "▶" button on the main menu.
2. **Dodge & Weave**: Avoid the red metro trains at all costs!
3. **Collect**: Grab every coin and gem you see to maximize your high score.
4. **Survive**: The game speeds up the longer you run. Stay focused!
